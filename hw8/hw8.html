Explanation of AJAX

First, it created a new XMLHttpRequest Object. Then, it created a funciton that will display the data in table format once the response from the server is ready. Called XMLHttpRequest Object's open("GET", "cd_catalog.xml") method to initialize the request to get the cd_catalog.xml. XMLHttpRequest Object send the request to the server. Once the server received the get request and sent back the requested data-cd_catalog.xml to the Object. The Object received the cd_catalog.xml, processed it and displayed the data on the page.
